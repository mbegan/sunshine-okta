# sunshine-okta

## Populate the config file
Make sure to grab the following:
1. Zendesk Subdomain
2. Zendesk Username
3. Zendesk Password
4. Okta Subdomain
5. Okta API Key

## Running it
```bash
pip install -r requirements.txt

python connector.py
```
